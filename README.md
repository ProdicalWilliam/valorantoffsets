# 🟣 ValorantOffsets

A **searchable browser for Valorant offsets and SDK references**, designed for reverse engineering and research. IDA parody, yes i know IDA isn't used to
look at decompiled SDK's but its a parody what do you expect?

## 🔹 Features

- Browse **Valorant SDK header files**  
- Search offsets quickly with **G-key hotkey**  
- User-friendly interface inspired by **IDA Pro**  
- Lightweight and fully **browser-based**

---

## 🔹 How It Works

1. Open the browser-based tool.  
2. Use the **search bar** (or press `G`) to quickly find functions or offsets.  
3. Switch between SDK Browser and Patch Offsets tabs using the **navigation tabs**.  
4. Explore files without needing any external software or IDA license.  

---

## 🔹 Keyboard Shortcuts

| Shortcut | Action |
|----------|-------|
| `G`      | Open Search |
| `Ctrl+F` | Find in File |
| `Esc`    | Close |
